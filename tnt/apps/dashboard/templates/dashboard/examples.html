Examples

There are a number of predefined templates that you can use to explore the different features of the TNTgo interface, which can be selected on the forst page of defining a new calculation.

Selecting a template will pre-fill the values in the TNTgo interface to simulate certain systems. If you are simply interested in exploring the capabilities of TNTgo, you can run them without making any changes. Alternatively you can use them as a convenient basis for your own calculations, and make changes such as adding Hamiltonian terms or explectation values to calculate, or changing the physical dimension or system size.

Spin: S=1 Heisenberg Chain

This sets up a similar calculation to that described in S.R. White and A.E. Feiguin (2004), but on a smaller scale due to the size limits in TNTgo.

First a ground state calculation is set up for a system with \(S=1\) and the Heisenberg Hamiltonian \[ H_g = \sum_{j=0}^{L-2} \left \{ \frac{1}{2}\left (\hat{S}_j^+ \hat{S}_{j+1}^- + \mathrm{h.c.}\right ) + \hat{S}_j^z \hat{S}_{j+1}^z \right \}. \] Note the form of the terms in the Hamiltonian, which are of course equivalent to \( \hat{\mathbf{S}}_j\cdot\hat{\mathbf{S}}_{j+1} \). On the previous page, we had set quantum number symmetries to be turned on by setting a total spin of zero (since we know the ground state lies in this spin sector). This then requires the Hamiltonian be defined in terms of U(1) invariant operators only, i.e. in terms of hopping and \(z\)-component of spin, rather than as its equivalent of sum of coupling terms in the three axes.

We then use this ground state as a base state for time evolution, but first modifying it by applying the operator \(\hat{S}^+\) on a site 20. We do this by applying a site-wide operator, with the delta function chosen as the spatial function. The system is then evolved under the same Heisenberg Hamniltonian as defined above, and like White and Feiguin the local magnetisation is chosen as an observable to track. In addition, we choose to track the overlap with the ground state and initial state as a function of time.

As described in White and Feiguin, the localised wave packet at \(t=0\) spreads out at later times to give irregular oscillations. White and Feiguin are able to evolve to longer times due to a larger maximum internal dimension (\(\chi \) is 150, as opposed to 40 used in this calculation), and a larger system size (\(L\) is 200 as opposed to 40 used in this calculation). If you wish to try using the TNT library code to reproduce these system sizes, or go beyond them, please consider installing the TNT library code on your own machines.
