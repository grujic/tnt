The Tensor Network Theory (TNT) Library

Tensor Network Theory (TNT) provides efficient and accurate methods for simulating strongly correlated quantum systems. The TNT library contains highly optimised routines for manipulating tensors in the network. The Dieter Jaksch theory group has provided a web interface to allow for simulation of common Hamiltonian lattice types.
