Tensor Network Theory (TNT) provides efficient and accurate methods for simulating strongly correlated quantum systems. TNTgo is a web interface for simulation of common Hamiltonian lattice types, provided by the Dieter Jaksch theory group. It is based on the TNT library, which contains highly optimised routines for manipulating tensors in the network.
